Les tags d'ouverture et de fermeture

Le début et la fin des portions de code PHP sont signalés grâce à des balises d’ouverture et de fermeture. Seul ce qui est entre ces balises (ou délimiteurs) est interprété par PHP, le reste est envoyé tel quel au navigateur.

Malgré l'existance de plusieurs variantes, il est primordial d'utiliser pour déclarer le PHP la balise <?php et pour terminer le script PHP la balise ?> le tag de fermuture est uniquement nécessaire si le code ne contient pas du Html ou du JavaScript en dessous du script PHP

Insertion de PHP dans HTML

Le code PHP peut être directement intégré dans les fichiers HTML. Il peut figurer à différents endroits de ces fichiers, tout en étant entrecoupé de code HTML.

<!doctype html>
<html lang="fr">
<head>
<meta charset="UTF-8">
<title>Les délimiteurs PHP</title>
</head>
<body>
<h1>PHP et le code HTML</h1>
<?php
$version = 'PHP7.2';
echo '<h2>La dernière version est '.$version.'</h2>';
?>
</body>

Les commentaires

Comme avec le langage C, les commentaires sont introduits par la séquence /* et se terminent par */. Par ailleurs, PHP utilise également les signes de commentaires //, qui permettent de commenter une ligne complète. Les commentaires ne sont pas interprétés et n'apparaissent pas dans le navigateur

Vous pouvez donc profiter des commentaires pour ajouter des explications sur les parties de code complexes afin d’en faciliter la relecture et la compréhension par la suite (maintenabilité du projet).

<?php
/*
Commentaires sur plusieurs lignes
Ici la deuxième ligne
*/
// Commentaires sur une ligne
echo "ceci n’est pas commenté" ;
?>
