Diagramme (IF)
Principe

L’instruction if(){} est la structure de test la plus basique. Elle permet d’exécuter une suite d’instructions en fonction d’une condition. La condition entre parenthèses est évaluée et les instructions situées entre les accolades sont exécutées seulement si l’évaluation aboutit à VRAI (TRUE)

Syntaxe:

            if(expression) {
                instruction;
            }

Exemple:

            if ($a > $b) {
              echo 'a est plus grand que b';
            }

La condition peut être complexe. Par exemple pour tester si l'âge d'une personne est compris entre 18 et 35 ans. Il est possible de combiner les deux expressions grâce à l’opérateur && pour en faire une expression unique qui ne renverra TRUE que si les deux composantes renvoient TRUE

            $age = 25;
            if($age > 18 && $age < 35) {
                 echo 'Votre profil est : jeune adulte';
            }
Diagramme (IF ELSE)
Principe

Lorsque vous voulez exécuter une instruction si une condition est remplie, et une autre instruction si cette condition n'est pas remplie vous devez ajouter une structure else{} après le if().

Syntaxe:

            if(condition){
                instructions si la condition est vérifiée.
            }
            else{
                instructions si la condition n’est pas vérifiée.
            }

Exemple:

            $temps = 'moche';
            if($temps == 'ensoleillé'){
                echo 'Il fait beau';
            }
            else{
                echo 'Il ne fait pas beau';
            }
Exemple avec du HTML et plusieurs délimiteurs 
            <?php if (condition): ?>
                Code HTML si la condition est vraie
            <?php else: ?>
                Code HTML si la condition est fausse
            <?php endif ?>
