Les variables

Principe

Les variables sont destinées à stocker les données qui seront utilisées et pourront être modifiées lors de l’exécution du programme. Il est important de noter que la plupart du temps les données des variables proviennent de l'extérieur du script (formulaire, base de données, sessions, cookies...)
Contrairement à d'autres langages, le PHP n’exige pas la déclaration de type des variables avant leur utilisation. Une variable peut prendre une valeur de chaîne de caractères et, ensuite, être utilisée pour contenir un entier (transtypage automatique).

Chaque variable possède un identifiant particulier, qui commence toujours par le caractère dollar ($) suivi du nom de la variable. Les règles de création des noms de variable sont les suivantes :
- Le nom commence par un caractère alphabétique, pris dans les ensembles [a-z], [A-Z] ou par le caractère de soulignement.
- Les chiffres peuvent être utilisés à l'exception du premier caractère du nom de la variable.
- Les noms des variables sont sensibles à la casse.

Conventions d'écriture: écrire vos variables en minuscules avec des noms évocateurs en employant le style "camelcase" dans le cas ou une variable nécessite plusieurs mots

            $name // valide
            $01name // invalide
            $_name // valide
            $NAME // valide
            $first name // invalide
            $first*name // invalide
            $firstName // valide
Les opérateurs d'assignation ou d'affectation

Les opérateurs sont des symboles qui permettent de manipuler des variables. Ils permettent notamment d’effectuer des opérations, d’affecter ou de comparer des valeurs, etc.

L’affectation consiste à donner une valeur à une variable. L’affectation se fait à l’aide de l’opérateur = en même temps que la création de la variable.

            syntaxe:
            $mavar = expression;

            exemples:
            $name = 'Servais';
            $age = 25;
            $connect = TRUE;

En plus du simple opérateur d'affectation, il existe des "opérateurs combinés" pour tous les opérateurs arithmétiques, l'union de tableaux et pour les opérateurs sur les chaînes de caractères. Cela permet d'utiliser la valeur d'une variable dans une expression et d'affecter le résultat de cette expression à cette variable.

           $montant = 100;
           $montant += 21; // affecte la valeur 21 à la variable $montant (correspond à l'instruction $montant = $montant + 21 )
           
           Remarque: Pour les opérateurs combinés vous pouvez utiliser tous les opérateurs arithmétiques ou de concaténation
            $quantite = 10;
            $quantite -= 2; // Deux produits offerts
            $quantite // 8
           
           $message = 'Erreur de type: ';
           $message .= 'Passorwd'; // affecte la valeur "Password" à la variable $message (correspond à $message = $message. 'Password')

Les opérateurs logiques

Les opérateurs logiques vont servir à faire des tests complexes. Ils sont principalement utilisés en combinaison avec certaines structures du langage (if, for, while, etc.) que nous verrons peu après. Ces opérateurs permettent de comparer deux termes et de renvoyer un booléen vrai ou faux (TRUE ou FALSE) selon la véracité de la comparaison.

Exemple Nom Résultat
$a and $b And (Et) TRUE si $a ET $b valent TRUE
$a or $b Or (Ou) TRUE si $a OU $b valent TRUE
$a xor $b XOR TRUE si $a OU $b est TRUE, mais pas les deux en même temps.
! $a Not (Non) TRUE si $a n'est pas TRUE
$a && $b And (Et) TRUE si $a ET $b sont TRUE
$a || $b Or (Ou) TRUE si $a OU $b est TRUE
La portée des variables

La portée d'une variable dépend du contexte dans lequel la variable est définie. Pour la majorité des variables, la portée concerne la totalité d'un script PHP. Mais, lorsque vous définissez une fonction, la portée d'une variable définie dans cette fonction est locale à la fonction. Par exemple : 

            $montant = 100;
            $q = 5;
            function total(){
                global $montant, $q;
                echo $montant * $q;
            }
            total();
