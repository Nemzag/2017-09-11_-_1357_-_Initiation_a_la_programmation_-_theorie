Les instructions de sortie

Afficher du contenu avec echo

echo — Affiche une chaîne de caractères

echo n'est pas vraiment une fonction (c'est techniquement une structure du langage), cela fait que vous n'êtes pas obligé d'utiliser des parenthèses. echo (contrairement à d'autres structures de langage) ne se comporte pas comme une fonction, il ne peut donc pas être utilisé dans le contexte d'une fonction. De même, si vous voulez passer plusieurs paramètres à echo, les paramètres ne doivent pas être entourés de parenthèses.

Quelques exemples avec des simple quotes

                echo '<h1>IEPSCF-NAMUR</h1>';// Commentaire mono-ligne
                echo 'l\'Etablissement propose plus de 100 formations';
                echo '<br>';
                echo "l'Etablissement propose plus de 100 formations";
                echo "il a dit: \"j'aime le Web\"";
                echo '<br>';

Les guillemets simples annulent le remplacement des variables. Vous devez les concatener avec l'opérateur de concaténation point(.)

                $name = 'Sophie'; // Déclaration et affectation de la variable
                echo 'Bienvenue '.$name.' dans votre panier'; // Substitution (utilisation) de la variable

Quelques exemples avec des double quotes

            echo "l'Etablissement propose plus de 100 formations";
            echo "Bienvenue $name dans votre panier"; // Remplacement de la variable sans concaténation

echo dispose aussi d'une version courte, où vous pouvez faire suivre la balise PHP ouvrante d'un signe égal (=).

Exemple

                <?php
                $name = 'John Doe';
                ...
                ...
                ?>
                <h3>IEPSCF</h3>
                <p>Enseignant: <?=$name?></p>
Afficher du contenu avec print

print — Affiche une chaîne de caractères

Tout comme echo, print n'est pas une fonction et ne nécessite pas de parenthèses

Quelques exemples

        print '<h1>IEPSCF-NAMUR</h1>';
        print "<h1>IEPSCF-NAMUR</h1>";
        print "Bienvenue $name dans votre panier";
